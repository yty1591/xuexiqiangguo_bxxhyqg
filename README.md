#### 介绍
本软件可帮助你自动学习，学习分数可达每日40-42分，可创建定时任务，每天定时学习
支持多账号自动学习，支持微信与邮箱消息推送，学习完成后自动提醒，自带防检测模块
不出滑块，无需人工操作，放心使用
支持新版学习强国2.57.0版本
